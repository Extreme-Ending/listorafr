1. What Listora provides

Listora is an SEO audit tool for Etsy sellers. You submit a public Etsy listing URL and receive a scored SEO report containing specific, listing-level recommendations on how to improve your title, tags, and description. The Service uses the Etsy Open API v3 to retrieve public listing data and an AI layer (Anthropic Claude API) to generate the written recommendations in your report.

The recommendations Listora provides are informational and educational. They are based on analysis of your listing's publicly available fields against observed SEO best practices for the Etsy marketplace. Listora does not guarantee any specific outcome including improved search ranking, increased traffic, higher conversion rates, or increased sales as a result of following the audit recommendations.

Listora accesses Etsy listing data exclusively through the Etsy Open API v3 under an active Etsy developer application. Listora's use of the Etsy API is governed by Etsy's API Developer Terms of Use in addition to these Terms.

Listora requests only the minimum data necessary to perform the SEO audit: listing title, tags, description, taxonomy, price, and image count. Listora does not request access to private shop data, buyer data, payment data, or any data scope beyond what is required for the stated purpose. All Etsy API calls are made on behalf of the seller requesting the audit, for the purpose of improving that seller's own listing.

Etsy listing data retrieved via the API is used solely to generate the audit report for the requesting seller. It is not sold, redistributed, aggregated into third-party databases, or used for any commercial purpose other than delivering the audit result to you.
