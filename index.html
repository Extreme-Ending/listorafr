Listora

A personal tool built to analyze the SEO performance of my own Etsy shop listings. It uses read-only API access to review listing titles, tags, and descriptions and identify specific optimization opportunities such as tag completeness and title structure.

About this tool

A personal project for improving my own shop listings.

Personal use only

Listora is built to analyze listings from my own Etsy shop. It is not a public service and does not process listings from other sellers' shops.

Read-only access

The tool only reads listing data such as titles, tags, and descriptions. It does not write, edit, or modify any content on the shop. No orders, payments, or customer data are ever accessed.

Processed internally

All listing data is processed within the application to identify SEO gaps such as unused tag slots and title structure issues. No listing data is shared with any third party.

Data use

Read-only. Own shop only. Nothing shared.

Listora uses the Etsy Open API v3 with read-only access limited to my own shop. The only data fetched is what is needed to run the SEO analysis: listing titles, tags, and descriptions. This data is processed entirely within the application and is not stored, sold, or shared with any third party.

The tool does not access orders, payment information, customer details, or any private shop data beyond the listing fields required for the analysis.
