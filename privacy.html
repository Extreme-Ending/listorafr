Privacy Policy

Listora is a personal tool built to analyze the SEO performance of my own Etsy shop listings. This policy explains how data is handled when the tool is in use.

What data is accessed

Listora uses the Etsy Open API v3 with read-only access to fetch listing titles, tags, and descriptions from my own Etsy shop. No other data is requested. The tool does not access orders, payment information, customer details, or any private shop analytics.

How data is used

Listing data is processed entirely within the application to identify SEO optimization opportunities such as tag completeness and title structure. The data is not stored beyond the active session, not sold, and not shared with any third party.

Scope

This tool is for personal use only and is limited to my own shop listings. It does not process or store listing data from any other Etsy seller.

Listora is an independent tool and is not affiliated with, endorsed by, or certified by Etsy, Inc. "Etsy" is a trademark of Etsy, Inc.
